# API Документация L_Shop

Базовый URL: `http://localhost:3000/api`

## Авторизация

### POST /auth/register

Регистрация нового пользователя.

**Тело запроса:**
```json
{
  "name": "Иван Иванов",
  "email": "ivan@example.com",
  "login": "ivan123",
  "phone": "+375291234567",
  "password": "secret123"
}
```

**Ответ 201:**
```json
{
  "message": "Регистрация успешна",
  "user": {
    "id": "uuid",
    "name": "Иван Иванов",
    "email": "ivan@example.com"
  }
}
```

**Ошибки:**
- 400 - Отсутствуют поля / Неверный email / Неверный телефон
- 409 - Пользователь уже существует

---

### POST /auth/login

Вход в систему.

**Тело запроса:**
```json
{
  "login": "ivan123",  // login или email
  "password": "secret123"
}
```

**Ответ 200:**
```json
{
  "message": "Вход выполнен",
  "user": {
    "id": "uuid",
    "name": "Иван Иванов",
    "email": "ivan@example.com"
  }
}
```

**Ошибки:**
- 400 - Отсутствуют учетные данные
- 401 - Неверные учетные данные

---

### POST /auth/logout

Выход из системы.

**Ответ 200:**
```json
{
  "message": "Выход выполнен"
}
```

---

### GET /auth/me

Получение информации о текущем пользователе.

**Требуется:** Authorization (sessionToken cookie)

**Ответ 200:**
```json
{
  "id": "uuid",
  "name": "Иван Иванов",
  "email": "ivan@example.com",
  "login": "ivan123",
  "phone": "+375291234567"
}
```

**Ошибки:**
- 401 - Не авторизован / Сессия истекла
- 404 - Пользователь не найден
