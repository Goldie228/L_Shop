# Чеклист дизайн-проверки

> Обязательный чеклист для проверки соответствия дизайну при code review

## Общие требования к UI/UX

### ✅ Соответствие дизайн-системе

- [ ] **Цветовая палитра**: используются CSS-переменные из `design-tokens.css`
  - Primary: `var(--color-primary-*)`
  - Secondary: `var(--color-secondary-*)`
  - Semantic: `var(--color-success-*)`, `var(--color-error-*)`, `var(--color-warning-*)`, `var(--color-info-*)`
  - Нет hardcoded цветов (например, `#dc2626` вместо `var(--color-error-600)`)

- [ ] **Типографика**: используются токены типографики
  - Размеры: `var(--font-size-*)`
  - Вес: `var(--font-weight-*)`
  - Высота строки: `var(--line-height-*)`
  - Шрифт: `var(--font-family-primary)`

- [ ] **Отступы**: используется 8px grid система
  - `var(--spacing-*)` для padding/margin
  - Семантические алиасы: `var(--spacing-xs)`, `var(--spacing-sm)`, `var(--spacing-md)`, `var(--spacing-lg)`, `var(--spacing-xl)`

- [ ] **Тени (elevation)**: используются токены теней
  - `var(--shadow-1)` - `var(--shadow-6)` для разных уровней
  - `var(--shadow-focus)` для состояния фокуса

- [ ] **Скругления**: используются токены радиусов
  - `var(--radius-*)` для border-radius

### ✅ Состояния компонентов

- [ ] **Hover**: плавный переход, изменение цвета/тени
  - Время: `var(--duration-fast)` (150ms)
  - Функция: `var(--ease-out)`

- [ ] **Focus**: видимое состояние фокуса
  - Кольцо фокуса: `var(--shadow-focus)` или `outline`
  - Доступность: `focus-visible` для клавиатурной навигации

- [ ] **Active**: состояние нажатия
  - Изменение цвета (темнее)
  - Transform: `scale(0.98)` (опционально)

- [ ] **Disabled**: отключённое состояние
  - Opacity: `0.6`
  - Cursor: `not-allowed`
  - Pointer events: `none`

- [ ] **Loading**: индикатор загрузки (если применимо)
  - Spinner или skeleton
  - Заблокированное взаимодействие

- [ ] **Error**: визуальное отображение ошибок
  - Цвет границы: `var(--color-error-*)`
  - Сообщение об ошибке видимо

### ✅ Адаптивность

- [ ] **Mobile-first**: стили для мобильных устройств по умолчанию
- [ ] **Breakpoints**: корректное использование медиа-запросов
  - Mobile: `< 640px`
  - Tablet: `640px - 1023px`
  - Desktop: `≥ 1024px`

- [ ] **Адаптивные компоненты**:
  - Header: hamburger menu на мобильных
  - ProductList: 1 колонка mobile, 2 tablet, 3-4 desktop
  - Cart: mobile (1 колонка), desktop (2 колонки)

- [ ] **Область касания**: минимум 44x44px для интерактивных элементов на мобильных

### ✅ Тёмная тема

- [ ] **CSS-переменные**: компонент корректно отображается в тёмной теме
- [ ] **Контрастность**: соответствие WCAG AA (4.5:1 для текста)
- [ ] **Тестирование**: проверено с `prefers-color-scheme: dark` или `.dark-theme`

### ✅ Анимации

- [ ] **Длительности**: используются стандартные значения
  - Fast: 150ms (hover, focus)
  - Normal: 250ms (открытие/закрытие)
  - Slow: 350ms (сложные трансформации)

- [ ] **Производительность**: анимируются только `transform` и `opacity`

- [ ] **prefers-reduced-motion**: анимации отключаются при соответствующей настройке

## Примеры типичных ошибок

| Ошибка | Как исправить |
|--------|---------------|
| `color: #3b82f6` | Заменить на `var(--color-primary-500)` |
| `margin: 10px` | Заменить на `var(--spacing-2)` (16px) или `var(--spacing-1)` (4px) |
| `border-radius: 5px` | Заменить на `var(--radius-2)` (6px) |
| Отсутствует hover | Добавить `:hover` с изменением цвета/тени |
| Нет focus-visible | Добавить `:focus-visible` с `var(--shadow-focus)` |
| Hardcoded размер | Использовать responsive единицы (%, vw/vh) |
